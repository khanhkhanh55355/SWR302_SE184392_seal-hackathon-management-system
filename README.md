# SEAL Hackathon Management System - Full Frontend Prototype

This is a complete React + Vite frontend prototype using mock data and LocalStorage.

## Main roles

- Team Leader / Member
- Mentor
- Judge
- Event Coordinator
- System Admin

## Features included

- Public home page
- Events page
- Public leaderboard
- Login / Register
- Role-based dashboard
- Register Team
- Submit Project
- View Results
- Track Team Progress
- Provide Feedback
- Complete Calibration Round
- Score Submissions
- Flag Conflict of Interest
- Configure Hackathon / Rounds
- Assign Judges
- Publish Leaderboard
- Manage Users
- View Audit Logs
- Data Overview
- LocalStorage mock database
- Reset mock data button
- Responsive web layout

## How to run

```bash
npm install
npm run dev
```

Open the link shown in the terminal, usually:

```bash
http://localhost:5173
```

## Notes

This project does not require a backend server. All data is stored in browser LocalStorage.
Use the "Reset Mock Data" button in the sidebar to restore the initial data.
